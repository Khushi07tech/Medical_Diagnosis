## Project Background
As someone transitioning from pre-med to computer science, I built this system to address healthcare accessibility challenges. This tool helps users get preliminary diagnosis suggestions based on their symptoms, particularly useful in underserved areas where immediate medical consultation isn't available.

## Features
- Symptom-based diagnosis matching
- Disease classification (Infectious, Chronic, Acute)
- Severity assessment
- Treatment recommendations
- Emergency detection
- Patient consultation history

## Tech Stack
- Python 3.x
- Object-Oriented Programming (Inheritance, Polymorphism)
- Pattern matching algorithms
